How to Convert ICS to CSV on Windows 10 (2025)

Published: April 2025 | Updated: April 2025 | Author: Tech Guides Team

Converting ICS to CSV on Windows 10 does not require installing any additional software. You can use a free online converter tool directly in your browser, or use Windows 10's built-in tools for a fully offline conversion. This guide covers both methods.

What You Need

Before starting, make sure you have the following:

  • An ICS file exported from Google Calendar, Outlook, or Apple Calendar
  • Microsoft Edge, Chrome, or Firefox on Windows 10
  • Microsoft Excel (optional, for viewing the CSV output)

Step 1 — Locate Your ICS File on Windows 10

Open File Explorer and navigate to your Downloads folder. If you exported from Google Calendar, you will find a ZIP file named after your Google account. Right-click the ZIP file and select Extract All. Choose a destination folder and click Extract.

Inside the extracted folder you will find one or more files with the .ics extension. Each file corresponds to a separate calendar. If you have multiple calendars in Google Calendar — personal, work, birthdays — each will have its own .ics file. Identify the one you want to convert.

Step 2 — Open a Free ICS to CSV Converter

Open Microsoft Edge or any browser on Windows 10 and navigate to a free ICS to CSV converter tool. No software installation is needed — the conversion runs entirely in your browser. This means your calendar data does not need to be sent to any external server, depending on the tool you choose.

Look for a converter that explicitly supports all ICS fields including DTSTART, DTEND, SUMMARY, DESCRIPTION, LOCATION, and RRULE. Some basic converters only extract the event title and dates, discarding other fields.

Step 3 — Upload and Convert the ICS File

On the converter tool's page, click the Choose File or Upload button. Windows 10 File Explorer will open — navigate to your extracted .ics file and select it. Click Open to upload the file to the converter.

The converter will parse the ICS file and display a preview of the extracted events. Check that the event count matches what you expect — if you have 50 events in your calendar, the preview should show 50 rows. Click Convert or Download CSV to generate and save the CSV file.

Step 4 — Download and Open the CSV File

The converted CSV file will download to your Windows 10 Downloads folder. To verify the conversion, open the CSV file in Microsoft Excel. Excel will automatically detect the CSV format and display your calendar events in rows and columns.

Check the following in the output:

  • Event titles appear in the correct column
  • Dates are formatted correctly (not showing as raw UTC timestamps)
  • All events from the original ICS file are present
  • Special characters in event descriptions display correctly

Method 2 — Convert ICS to CSV Using Windows 10 PowerShell

For users who prefer an offline method or need to process multiple ICS files, Windows 10 PowerShell can be used to convert ICS to CSV without any browser or internet connection.

Open PowerShell on Windows 10 by pressing Win + X and selecting Windows PowerShell. The conversion requires parsing the ICS text format and extracting the relevant fields. This approach gives you full control over which ICS properties to include in the output CSV and how to handle time zones and recurring events.

Method 3 — Convert ICS to CSV Using Python on Windows 10

Python is available for free on Windows 10 via the Microsoft Store. After installing Python, you can use the icalendar library to parse ICS files and the built-in csv module to write the output. Install the library by running pip install icalendar in Command Prompt. This method is the most flexible and supports all ICS fields including recurring events and time zone handling.

Common Issues on Windows 10

ICS File Shows as Unknown File Type

Windows 10 may not recognize the .ics file extension if no calendar application is installed. This does not affect the conversion — you can still upload the file to an online converter or process it with PowerShell or Python regardless of the file association.

CSV Opens as a Single Column in Excel

If Excel displays all data in a single column instead of separate columns, the CSV delimiter may not be recognized. Go to Data → Text to Columns, select Delimited, check Comma, and click Finish to split the data correctly.

Date Fields Show Incorrect Times

ICS files store times in UTC by default. If your converted CSV shows times that are offset from your local time, the converter may not be applying your local time zone offset. Look for a converter that lets you specify your time zone during conversion.

Frequently Asked Questions

Does Windows 10 have a built-in ICS to CSV converter?

No. Windows 10 does not include a native ICS to CSV conversion tool. You need either a free online converter, PowerShell, or a third-party application.

Is it safe to upload ICS files to an online converter?

Your ICS file contains calendar event data including meeting titles, locations, and descriptions. Use a reputable converter and avoid uploading files containing sensitive or confidential meeting information to unknown services.

Can I convert multiple ICS files at once on Windows 10?

Most online converters process one file at a time. For batch conversion of multiple ICS files, the PowerShell or Python methods are more suitable as they can be scripted to process entire folders of ICS files automatically.

What is the difference between ICS and vCard?

ICS (iCalendar) stores calendar events while vCard (.vcf) stores contact information. They are both plain text formats but serve different purposes and are not interchangeable.
